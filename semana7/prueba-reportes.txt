Prueba de la API de reportes comunitarios:

1. Al enviar POST a http://localhost:3000/reportes con datos, el servidor guarda el reporte en el array "reportes" y responde con un mensaje de confirmación y el reporte creado.

2. Al consultar GET en http://localhost:3000/reportes, el servidor devuelve todos los reportes guardados en formato JSON.

Ejemplo de respuesta GET /reportes:
[
  {
    "id": 1,
    "tipo": "Infraestructura",
    "descripcion": "Daño en alumbrado público"
  },
  {
    "id": 2,
    "tipo": "Ambiente",
    "descripcion": "Acumulación de residuos en una zona comunitaria"
  }
]

Nota: Los datos se guardan en memoria. Si se reinicia el servidor, la lista se borra porque aún no usamos base de datos.
